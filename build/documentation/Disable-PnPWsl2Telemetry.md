# Disable-PnPWsl2Telemetry

## SYNOPSIS
Disables the telemetry for the PnP.WSL2 module.

## SYNTAX

```
Disable-PnPWsl2Telemetry
```

## DESCRIPTION
The Disable-PnPWsl2Telemetry function is used to disable telemetry for PnP.WSL2 .



## EXAMPLE 1
```
Disable-PnPWsl2Telemetry
```
