# Stop-PnPWsl2

## SYNOPSIS
Stops the WSL2 service.

## SYNTAX

```
Stop-PnPWsl2 [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
The Stop-PnPWsl2 cmdlet stops the WSL2 service if it is currently running.



## EXAMPLE 1
```
Stop-PnPWsl2
```

This command stops the WSL2 service.
