    PnP Wsl2 Candy

    PnP Wsl2 has around 21 Candy (tools/scripts) which can help you in setting up, configuring, maintaining and using various WSL instances. Each of the cmdlets is documented to aid in learning how to use it.

    Azure

    • ub-az-AzureCli

      Installs Azure Cli, Microsoft Azure command-line interface used to create and manage Azure resources

    • ub-az-AzureCli+CreatePAT

      Installs Azure DevOps utils : includes Azure Cli and CreatePAT candy ( creates a AzureDevOps Personal Access Token (1 Year))

    • ub-az-CreatePAT

      Creates a AzureDevOps Personal Access Token that is valid for 1 year

    • ub-az-SSHKeyAddToAzDevops

      This script is used to add a WSL SSH key to an Azure DevOps organization.

    It prompts the user to enter the Azure DevOps organization and the SSH key name; Gets the SSH key from the .ssh directory; Adds the SSH key to the Azure DevOps organization;

    m365

    • ub-m365-PnPCliMicrosoft365

      Installs @pnp/cli-microsoft365.

    • ub-m365-PnPPowerShell

      This script will install PnP.PowerShell

    • ub-m365-PSCore+PnPPowerShell

      This script will install PSCore and PnP.PowerShell

    Node

    • ub-node-Install

      This script installs a specified version of Node.js using the Node Version Manager (NVM). It first checks if the requested version is already installed. If the requested version is not installed, the script installs it using NVM and then sets it as the current version. The version number to be installed should be passed as an input .

    • ub-node-Nvm

      This script installs the Node Version Manager (nvm)

    • ub-node-RimRaf

      This script installs the RimRaf Node tool (deep deletion module for node)

    pwsh

    • ub-pwsh-PowerShellCore

      This script installs the latest version of PowerShell Core

    Python

    • ub-python-miniconda

      Installs Miniconda (small bootstrap version of Anaconda:includes conda, Python, usefull pkgs),Tensorflow (open source machine learning platform), DirectML (high-performance, hardware-accelerated DirectX 12 library for machine learning)

    SPFX

    • ub-spfx-FullDevEnvironment

      This script will install all needed assets for SharePoint Framework Development environment following Microsoft guidance mentioned here :https://learn.microsoft.com/en-us/sharepoint/dev/spfx/set-up-your-development-environment

      1. It will install nvm , node (you can select the version), gulp, yeoman ,microsoft yeoman generator, and spfxfastserve
      2. Will create the sfpx self-signed developer certificate,updating it to your windows host
      3. Will also create the sshkey in wsl updating it to your windows host and add it to your Azure Devops instance

    • ub-spfx-git-Config

      This script is used to configure git (username;useremail).

    • ub-spfx-gulp-TrustDevCert

      This script is used to install the SPFX Dev Cert.

    Gulp+Yeoman+MicrosoftGen+SpfxFastServe

    • ub-spfx-gulp-Yo-Mgen-Fs

      This script is used to install gulp-cli yo @microsoft/generator-sharepoint spfx-fast-serve.

    SYS

    • ub-sys-wsl-Initialize

      Initializes a ubuntu environment

    • ub-sys-wsl-SSHKeyGenerate

      This script is a bash shell script that will create an SSH key in WSL and will update your windows host with the same key

    • ub-sys-wsl-SuperUser

      This script is a bash shell script that sets the default user for a WSL instance

    • ub-sys-wsl-Updates

      This script is a bash shell script that updates the current Linux distribution packages

    addOns

    • ub-sys-wsl-Utils

      This script is a bash shell script that installs utilities for WSL (Linux Subsystem for Windows)
