    PnP Wsl2 Candy

    PnP Wsl2 has around 21 candies (tools/scripts) which can help you in setting up, configuring, maintaining and using various WSL instances. Each of the cmdlets is documented to aid in learning how to use it.

    Azure

    • ub-az-AzureCli

      This script will install AzureCli.

    • ub-az-AzureCli+CreatePAT

      Installs AzureDevops utils (includes Azure Cli+ creating a AzureDevOps Personal Access Token)

    • ub-az-CreatePAT

      Creates a AzureDevOps Personal Access Token that is valid for 1 year

    SPFX

    • ub-spfx-FullDevEnvironment

    • ub-spfx-git-Config

      This script is used to configure git (username;useremail).

    • ub-spfx-gulp-TrustDevCert

      This script is used to install the SPFX Dev Cert.

    Gulp+Yeoman+MicrosoftGen+SpfxFastServe

    • ub-spfx-gulp-Yo-Mgen-Fs

      This script is used to install gulp-cli yo @microsoft/generator-sharepoint spfx-fast-serve.

    Node

    • ub-node-Install

    Python

    • ub-python-miniconda

      Installs Miniconda (small bootstrap version of Anaconda:includes conda, Python, usefull pkgs),Tensorflow (open source machine learning platform), DirectML (high-performance, hardware-accelerated DirectX 12 library for machine learning)

    Node

    • ub-node-Nvm

      This script installs the Node Version Manager (nvm)

    m365

    • ub-m365-PnPCliMicrosoft365

      Installs @pnp/cli-microsoft365.

    • ub-m365-PnPPowerShell

      This script will install PnP.PowerShell

    pwsh

    • ub-pwsh-PowerShellCore

      This script installs the latest version of PowerShell Core

    m365

    • ub-m365-PSCore+PnPPowerShell

      This script will install PSCore and PnP.PowerShell

    Node

    • ub-node-RimRaf

      This script installs the RimRaf Node tool (deep deletion module for node)

    AzDevOp

    • ub-az-SSHKeyAddToAzDevops

    SYS

    • ub-sys-wsl-Initialize

      Initializes a ubuntu environment

    • ub-sys-wsl-SSHKeyGenerate

      This script is a bash shell script that will create an SSH key in WSL and will update your windows host with the same key

    • ub-sys-wsl-SuperUser

      This script is a bash shell script that sets the default user for a WSL instance

    • ub-sys-wsl-Updates

      This script is a bash shell script that updates the current Linux distribution packages

    addOns

    • ub-sys-wsl-Utils

      This script is a bash shell script that installs utilities for WSL (Linux Subsystem for Windows)
